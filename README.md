# DSA Pattern-Based Practice

A structured repository of my Data Structures & Algorithms (DSA) practice, focused on **understanding patterns, building intuition, and writing optimized solutions** — not just solving problems.

---

## Goals

-  Master core DSA patterns (not memorize solutions)
-  Build strong problem-solving intuition
-  Write clean, efficient, and scalable code
-  Maintain consistent practice across platforms

---

##  Repository Structure

Problems are organized **by patterns**, not by platform or difficulty


> This structure helps in **better revision and interview preparation**, since similar problems stay together.

---

##  Platforms Covered

- LeetCode  
- GeeksforGeeks  
- CSES  
- (More to be added)

Each problem mentions its source inside the file.

---

##  What Each Problem Contains

Each solution includes:

-  Approach explanation  
- Key insight (core idea behind the solution)  
- Mistakes / initial thought process  
- Time & Space complexity  
- Multiple approaches (Brute → Better → Optimal)

---
## Key Focus Areas

- Arrays & Two Pointers  
- HashMap & Frequency Tricks  
- Binary Search Patterns  
- Graph Traversals (BFS/DFS)  
- Dynamic Programming  
- Mathematical & Bit Manipulation
